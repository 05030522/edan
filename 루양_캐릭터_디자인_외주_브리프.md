# 루양(Lamby) 캐릭터 디자인 외주 브리프

> **프로젝트**: 에덴(Eden) - 기독교 성경 묵상 앱
> **슬로건**: "매일 5분, 에덴에서 만나요"
> **작성일**: 2026-03-26
> **버전**: v1.0

---

## 1. 프로젝트 개요

에덴은 **매일 성경 묵상과 기도를 습관화**하는 기독교 앱입니다.
사용자는 매일 말씀을 읽고, 묵상하고, 기도하면서 자신만의 **영적 정원**을 가꿔나갑니다.

루양은 이 여정에서 사용자와 함께하는 **마스코트이자 가이드 캐릭터**입니다.

### 앱의 핵심 키워드
- 경건함 + 귀여움 (Sacred & Cute)
- 성경 묵상, 기도, 영적 성장
- 정원 가꾸기 메타포 (씨앗 → 에덴동산)
- 10~30대 한국 기독교인 대상

## 2. 캐릭터 컨셉

### 2-1. 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 루양 (Lamby) |
| **모티브** | 어린 양 (성경의 양 이미지) |
| **성별** | 중성 / 무성별 |
| **성격** | 따뜻하고 다정함, 살짝 엉뚱함, 격려를 잘 해줌 |
| **역할** | 사용자의 영적 여정 동반자, 앱 내 가이드 |
| **톤앤매너** | 친근한 반말체 ("오늘도 함께해요~", "돌아왔구나!") |

### 2-2. 성경적 상징

- **어린 양**: 순수함, 신뢰, 목자(하나님)를 따르는 존재
- **에덴동산의 동물**: 평화로운 공존, 하나님과의 교제
- 종교적으로 무겁지 않되, 은은하게 성경적 모티브가 느껴지는 방향

### 2-3. 성격 키워드

```
따뜻함 / 순수함 / 격려 / 위로 / 엉뚱함 / 사랑스러움
```

- 아침 인사를 건네는 다정한 친구
- 묵상을 함께하는 조용한 동반자
- 스트릭이 깨졌을 때 위로해주는 따뜻한 존재
- 가끔 엉뚱한 말로 웃음을 주는 귀여운 캐릭터

## 3. 비주얼 디자인 가이드

### 3-1. 현재 코드 기반 참고 컬러

현재 앱 내에서 코드(CustomPaint)로 그려진 루양의 컬러입니다.
**디자이너의 해석을 환영**하되, 앱 전체 톤과의 조화를 고려해주세요.

| 부위 | 현재 색상 | 비고 |
|------|-----------|------|
| 몸/털 | Cream `#FFF8E7` | 부드럽고 따뜻한 크림색 |
| 귀/볼 | Pink `#FFB6C1` | 라이트 핑크 |
| 코/강조 | Gold `#E5C88E` | 앱 브랜드 골드 |

### 3-2. 앱 컬러 팔레트 (디자인 맥락)

루양이 놓이는 앱의 전체 색감입니다.

| 용도 | 색상 | HEX |
|------|------|-----|
| 메인 브랜드 | Eden Green | `#B8D4BE` |
| 보조 (축복) | Gold | `#E5C88E` |
| 액센트 (은혜) | Soft Pink | `#F2E2E2` |
| 명상 | Purple | `#D4D0E6` |
| 배경 (라이트) | Dawn Light | `#FDF6ED` |
| 배경 (다크) | Deep Navy | `#1A1A2E` |

### 3-3. 디자인 스타일 방향

```
스타일: 플랫 일러스트 / 미니멀 캐릭터
머리신체비율: 2~2.5등신 (귀여움 극대화)
선: 부드러운 라운드 선 또는 선 없는 면 위주
질감: 깨끗하고 단순, 필요시 살짝 텍스처 (울 느낌)
```

**레퍼런스 스타일 키워드** (Pinterest/Behance 검색용):
- 카카오프렌즈 / 라인프렌즈 느낌의 심플 캐릭터
- Molang, Sumikko Gurashi 같은 동글동글 귀여운 동물 캐릭터
- 기독교 앱이지만 "교회 스테인드글라스" 느낌은 지양
- 10~30대가 "귀엽다"고 느끼는 현대적 캐릭터

### 3-4. 피해야 할 것

- 지나치게 사실적인 양 묘사 (동물 일러스트 X)
- 종교적으로 무거운 상징 (십자가를 들고 있다거나)
- 유아용으로 보이는 지나친 유치함
- 날카로운 선, 각진 형태

## 6. 말풍선 & UI 연동

루양은 항상 **말풍선**과 함께 등장합니다.

```
  ┌─────────────────────────┐
  │  좋은 아침이에요!        │
  │  오늘도 말씀과 함께해요~ │
  └──────────┬──────────────┘
             ▼
          ┌─────┐
          │ 루양 │
          │(표정)│
          └─────┘
```

- 말풍선은 루양 위에 표시 (꼬리가 아래를 향함)
- 말풍선 디자인도 함께 제안해주시면 좋습니다
- 둥글둥글한 말풍선, 앱 톤과 맞는 스타일

## 10. 커뮤니케이션

### 피드백 프로세스
1. 러프 스케치 단계에서 **3안** 제출
2. 1안 선택 후 **컬러 시안** 진행
3. 컬러 확정 후 **표정 작업** 진행
4. 각 단계마다 **1~2회 수정** 포함

### 파일 공유
- Figma 또는 Google Drive 공유
- 중간 산출물은 PNG 미리보기로 확인
- 최종본은 원본 + 내보내기 파일 모두 납품
