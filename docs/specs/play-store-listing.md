## 그래픽 에셋 요구사항

| 에셋 | 크기 | 필수 | 비고 |
|---|---|---|---|
| 앱 아이콘 | 512×512 PNG | 필수 | 투명도 X, 이미 자동 생성됨 (`루양.png`) |
| 피처 그래픽 | 1024×500 PNG/JPG | 필수 | 루양 + "에덴" 로고 조합 |
| 스크린샷 (폰) | 최소 2장, 최대 8장 | 필수 | 1080×1920 권장 |
| 스크린샷 (7" 태블릿) | 최대 8장 | 선택 | |
| 스크린샷 (10" 태블릿) | 최대 8장 | 선택 | |
| 프로모션 동영상 | YouTube URL | 선택 | 30초~2분 |

### 추천 스크린샷 순서 (폰 8장)
1. 웰컴 화면 (루양 + 슬로건)
2. 홈 화면 (일일 태스크 3개 + 스트릭)
3. 묵상 읽기 화면 (본문 + 루양 말풍선)
4. 퀴즈 풀이 화면
5. 정원(프로필) 레벨 성장
6. 주간 퀘스트 진행도
7. 커뮤니티/랭킹
8. 업적 화면
