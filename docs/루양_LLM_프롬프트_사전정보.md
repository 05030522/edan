# 루양(Lamby) 캐릭터 생성을 위한 LLM 사전정보

> 이미지 생성 AI(Midjourney, DALL-E, Stable Diffusion 등)나
> 캐릭터 디자인 전문 LLM에 넣기 위한 종합 컨텍스트 문서

---

## 1. 프로젝트 정체성

### 앱 이름: 에덴 (Eden)
- **슬로건**: "매일 5분, 에덴에서 만나요"
- **카테고리**: 기독교 성경 묵상 & 영적 성장 앱
- **타겟 유저**: 한국 10~30대 기독교인
- **핵심 경험**: 매일 말씀 읽기 → 묵상 → 기도 → 정원이 자라남
- **디자인 철학**: "Sacred & Cute" (경건함 + 귀여움의 공존)

### 앱의 세계관
사용자는 자신만의 **영적 정원**을 가꾸는 여정에 있다. 매일 성경을 읽고 묵상하면 정원이 자라나고, 루양이라는 어린 양 캐릭터가 이 여정을 함께한다. 정원은 맨땅(Lv1)에서 시작해 에덴동산(Lv10)까지 성장한다.

## 2. 캐릭터 기본 설정

### 이름 & 정체
- **이름**: 루양 (Lamby)
- **종**: 어린 양 (Lamb)
- **성별**: 중성 / 무성별
- **나이**: 알 수 없음 (영원히 어린 느낌)
- **서식지**: 에덴동산 (앱 세계 안)

### 성경적 배경
- 성경에서 양은 **순수함, 순종, 목자(하나님)를 신뢰하는 존재**를 상징
- "나는 선한 목자라" (요한복음 10:11)의 양 이미지
- 종교적으로 무겁지 않게, **은은하게** 성경 모티브가 느껴지는 수준
- 십자가, 후광 등 직접적 종교 상징은 넣지 않음

### 성격
- **따뜻함**: 아침마다 먼저 인사를 건네는 다정한 존재
- **격려**: "잘하고 있어! 계속 함께해~" 같은 응원
- **위로**: 스트릭이 깨져도 "괜찮아~ 다시 시작하자!" 라고 말함
- **순수함**: 세상 물정 모르는 듯한 맑은 느낌
- **살짝 엉뚱함**: 가끔 뜬금없이 귀여운 말을 함
- **부지런함**: 정원을 가꾸고 책을 읽는 등 열심히 하는 모습

### 루양의 실제 대사 (톤 참고)
```
아침 인사:
- "좋은 아침이에요! 오늘도 말씀과 함께해요~"
- "반가워! 오늘은 어떤 말씀을 만날까?"
- "돌아왔구나! 함께 묵상하자~"
- "오늘 하루도 주님과 동행해요!"
- "에덴에 온 걸 환영해! 오늘도 파이팅!"

격려:
- "잘하고 있어! 계속 함께해~"
- "말씀이 마음에 심겨지고 있어!"
- "오늘도 한 걸음 더 성장했어!"
- "나도 이 말씀 좋았어~ 함께 읽으니까 더 좋다!"

위로 (스트릭 깨졌을 때):
- "괜찮아~ 하나님의 은혜는 새 날마다 새로워! 다시 시작하자!"
- "쉬어가는 것도 괜찮아. 에덴은 항상 여기 있어~"
- "포기하지 않은 것 자체가 대단해! 다시 함께하자~"
- "넘어져도 다시 일어나는 게 진짜 믿음이야!"
```

## 3. 비주얼 디자인 상세

### 3-1. 현재 코드 기반 구조 (정확한 형태 묘사)

현재 루양은 Flutter CustomPaint로 원과 타원의 조합으로 그려져 있다.

**몸체 구조:**
```
[머리] 원형 (전체 크기의 48% 직경)
  ├─ 머리 위에 양털 돌기 5개 (작은 원 5개가 반원형으로 배치)
  ├─ 머리 안에 얼굴 요소 배치
  │
[몸통] 원형 (전체 크기의 64% 직경), 머리보다 아래 배치
  ├─ 몸통 주변에 양털 돌기 7개 (작은 원이 몸 둘레를 따라 배치)
  │
[귀] 타원형 2개, 머리 좌우 위쪽에 약간 바깥으로 기울어짐
  ├─ 바깥: 크림색
  ├─ 안쪽: 핑크색
  │
[발] 작은 타원 2개, 몸통 아래 좌우
```

**얼굴 구조:**
```
[눈] 작은 원형 2개 (검정), 각각 안에 하이라이트(흰 점)
  ├─ 신난 표정일 때: 골드색 추가 반짝임
  ├─ 잠든 표정: 반원 아치형 (감은 눈)
  ├─ 슬픈 표정: 눈썹이 안쪽 위로 올라감
  ├─ 놀란 표정: 눈이 평소보다 30% 더 큼

[코] 작은 타원형, 진한 핑크색, 얼굴 중앙

[볼] 반투명 핑크 원 2개, 코 양쪽

[입] 표정에 따라 변화:
  ├─ 행복/신남/응원/정원/읽기: 미소 (아래로 볼록한 호)
  ├─ 생각: 작은 "o" 모양 (살짝 오른쪽)
  ├─ 잠듦: 일자 (─)
  ├─ 슬픔: 아래로 처진 호 (∪ 뒤집힘)
  ├─ 기도: 아주 작고 평화로운 미소
  ├─ 놀람: 동그란 "O" (반투명 핑크 채움)
```

## 4. 10가지 표정/포즈 상세

### 포즈 1: happy (행복/기본)
- **사용처**: 홈 화면 기본 상태
- **표정**: 반달 눈, 미소 입, 핑크 볼
- **포즈**: 정면, 팔은 자연스럽게 몸 옆
- **소품**: 없음
- **분위기**: 밝고 편안한 기본 상태

### 포즈 2: thinking (생각)
- **사용처**: 묵상 화면, 퀴즈 풀 때
- **표정**: 눈이 위를 향함, 입은 작은 "o" 모양 (살짝 오른쪽으로 치우침)
- **포즈**: 고개 살짝 기울임
- **소품**: 머리 위에 생각 말풍선 (작은 점 → 중간 점 → 물음표 원)
- **분위기**: "음~ 뭘까?" 하는 귀여운 고민

### 포즈 3: excited (신남)
- **사용처**: 레벨업, 보상 획득
- **표정**: 반짝이는 눈 (골드색 추가 하이라이트), 활짝 웃는 입
- **포즈**: 양팔 벌리거나 점프하는 느낌
- **소품**: 주변에 골드 반짝이 4개 (별 모양)
- **분위기**: "와~! 대단해!" 하는 기쁨

### 포즈 4: sleeping (졸림/수면)
- **사용처**: 늦은 시간(밤) 접속
- **표정**: 감은 눈 (반원 아치형), 일자 입
- **포즈**: 살짝 기울어진 채 졸고 있는 모습
- **소품**: 오른쪽 위로 "Zzz" 글자 (작은것→큰것 3개, 반투명)
- **분위기**: 평화롭고 고요한 수면

### 포즈 5: cheering (응원)
- **사용처**: 일일 과제 완료 시
- **표정**: 반짝이는 눈 (골드 하이라이트), 활짝 미소
- **포즈**: 양손에 작은 깃발이나 폼폼을 들고 흔드는 모습
- **소품**: 주변에 컨페티 12개 (핑크, 골드, 하늘색, 보라 — 원형과 사각형 혼합)
- **분위기**: "해냈어! 축하해!" 하는 열띤 응원

### 포즈 6: sad (슬픔)
- **사용처**: 스트릭 깨졌을 때
- **표정**: 축 처진 눈, 안쪽 위로 올라간 눈썹(반투명), 아래로 처진 입
- **포즈**: 몸이 살짝 움츠러든 느낌
- **소품**: 왼쪽 눈 아래에 투명한 눈물방울 1개 (하늘색)
- **분위기**: 속상하지만 곧 괜찮아질 것 같은 귀여운 슬픔 (무겁지 않게)

### 포즈 7: praying (기도)
- **사용처**: 기도 기능 사용 시
- **표정**: 평화롭게 감은 눈(sleeping과 유사), 아주 작고 은은한 미소
- **포즈**: 두 손을 모으고 기도하는 자세
- **소품**: 머리 위에 골드색 은은한 빛 (후광이 아닌 부드러운 글로우, 반투명 25%)
- **분위기**: 경건하고 평화로운 순간 (앱의 핵심 감성)

### 포즈 8: surprised (놀람)
- **사용처**: 특별 이벤트, 예상치 못한 보상
- **표정**: 평소보다 30% 큰 동그란 눈, 동그란 "O" 입 (반투명 핑크 채움)
- **포즈**: 몸이 살짝 뒤로 젖혀진 느낌
- **소품**: 머리 오른쪽 위에 골드색 느낌표 (!)
- **분위기**: "어머! 이게 뭐야?" 하는 깜짝 놀람

### 포즈 9: gardening (정원 가꾸기)
- **사용처**: 정원 화면
- **표정**: 행복한 미소, 반달 눈
- **포즈**: 한 손에 작은 물뿌리개나 삽을 들고 있는 모습
- **소품**: 옆에 핑크 꽃 1송이 (꽃잎 5장 + 골드 중심 + 초록 줄기와 잎)
- **분위기**: 정원을 가꾸는 즐거움, 부지런한 모습

### 포즈 10: reading (읽기)
- **사용처**: 말씀 읽기 화면
- **표정**: 행복한 미소, 집중하는 듯한 눈
- **포즈**: 앞에 작은 책을 펼쳐 들고 있는 모습
- **소품**: 갈색 표지의 펼쳐진 책 (좌우 흰 페이지, 왼쪽에 글줄 3개, 가운데 척추선)
- **분위기**: 조용히 말씀에 집중하는 평화로운 모습

## 5. 앱 내 사용 맥락

### 5-1. 홈 화면
```
┌─────────────────────────┐
│  [오늘의 말씀 카드]       │
│  [일일 퀘스트 목록]       │
│                         │
│  ┌─────────────────┐    │
│  │ (루양) "좋은 아침!"│    │  ← 루양 + 말풍선이 카드 안에
│  └─────────────────┘    │
└─────────────────────────┘
```
- 루양 아이콘(44x44px 원형) + 오른쪽에 "루양" 라벨 + 인사말 텍스트
- GlassCard 컴포넌트 안에 배치

### 5-2. 정원 화면
```
┌─────────────────────────┐
│  [그라디언트 배경]         │
│                         │
│      (장식 아이콘들)      │
│         (루양)           │  ← 정원 중앙, 280px 높이 영역
│        "루양"            │
│      (장식 아이콘들)      │
│                         │
│  [레벨: Lv5 자라는 나무]  │
│  [진행바]                │
└─────────────────────────┘
```
- 정원 중앙에 루양 배치 (현재 80~100px)
- 레벨에 따라 배경 그라디언트와 주변 장식 아이콘이 변화

### 5-3. 말풍선 시스템
```
  ┌─────────────────────────┐
  │  텍스트 내용              │
  │  (bodyMedium, 14px)      │
  └───────────┬─────────────┘
              ▼ (삼각형 꼬리)
           [루양]
```
- 말풍선: 둥근 모서리 16px, 최대 너비 260px
- 라이트모드: 흰 배경 + 어두운 글씨
- 다크모드: 어두운 배경 + 밝은 글씨
- 등장 애니메이션: 400ms 페이드인
- 자동 사라짐: 3초 (설정 가능)

### 5-4. 사용 크기 정리
| 위치 | 크기 | 비고 |
|------|------|------|
| 홈 인사 카드 | 44px (아이콘) | 원형 컨테이너 안 |
| 정원 중앙 | 80~100px | 배경 위 중앙 배치 |
| 말풍선 동반 | 60~80px | 말풍선 아래 |
| 로딩 화면 | 100~120px | 바운스 애니메이션 |
| 스플래시 | 200px+ | 앱 시작 화면 |

### 기본 Idle 애니메이션
- **바운스**: 위아래로 부드럽게 통통 뛰는 움직임
- **진폭**: 전체 크기의 4% (120px 캐릭터 → 약 5px 위아래)
- **주기**: 1.2초 (올라갔다 내려오는 1사이클)
- **이징**: easeInOut (부드러운 가감속)
- **사인파 기반**: sin(value * π)로 자연스러운 곡선

### 표정 전환
- 표정은 즉시 전환 (현재는 애니메이션 없음)
- 향후: Lottie 또는 스프라이트 시트로 부드러운 전환 가능

## 9. 상점 아이템과의 관계 (참고)

앱에는 상점이 있어서 사용자가 코스메틱 아이템을 구매할 수 있다. 루양과 직접 관련된 아이템은 아니지만, 디자인 톤이 일관되어야 한다.

**상점 아이템 카테고리:**
- 프로필 프레임 (골드/실버/초록 테두리)
- 칭호 (믿음의 새싹, 기도의 용사 등)
- 배경 (새벽 하늘, 별빛 등)
- 정원 아이템: 올리브 나무, 장미, 벤치, 분수, 전망대 등

**아이템 레어리티 색상:**
- Common: `#9E9E9E` (회색)
- Rare: `#42A5F5` (파랑)
- Epic: `#AB47BC` (보라)
- Legendary: `#FFA726` (주황/골드)
